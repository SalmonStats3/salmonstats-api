# API for Salmon Stats+

## Configuration

```.env
NODE_PORT=
DATABASE_URL=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_DB=

```

### Example

```env
NODE_PORT=8080
DATABASE_URL=postgresql://tkgstrator:1234@localhost:5432/splatoon3
POSTGRES_USER=tkgstrator
POSTGRES_PASSWORD=1234
POSTGRES_DB=splatoon3
API_VER=1.0.0
```
